# AWS Bedrock Customer Support Chatbot

An AI-powered **customer support chatbot** built using **AWS Bedrock**, **OpenAI**, and **Groq**.  
This project demonstrates **multi-model integration**, **tool routing**, and **conversation memory** for intelligent customer support automation.

---

## Features

- AI-powered customer support chatbot
- AWS Bedrock integration
- OpenAI integration
- Groq integration
- Simple model/tool routing
- Conversation memory support
- Extensible backend architecture
